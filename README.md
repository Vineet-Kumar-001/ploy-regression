# Polynomial Regression Project

## Overview

This repository demonstrates the implementation of **Polynomial Regression**, a supervised machine learning technique used to model non-linear relationships between independent and dependent variables.

Unlike linear regression, polynomial regression fits a curve to the data by introducing polynomial terms, allowing it to capture more complex patterns.

---

## Objective

* Understand the limitations of linear regression
* Learn how to transform features for non-linear modeling
* Implement polynomial regression using Python
* Visualize model performance

---
